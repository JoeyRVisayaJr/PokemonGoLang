# Pokémon Battle Simulator — Run the executable

To run the prebuilt Windows executable:

Open PowerShell in the project directory and run:

```powershell
.\pokemon-battle.exe
```

Then open your browser to:

http://localhost:8081

Notes:
- The executable must be in the current directory. If it's been removed by `.gitignore`, rebuild with `go build`.
- On first run Windows Defender/SmartScreen may prompt — choose to allow/rerun if you trust the binary.
